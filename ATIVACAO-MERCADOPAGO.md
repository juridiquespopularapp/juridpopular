# Como Ativar PIX e Cartões no MercadoPago ✅

## 1. Recebedores (Meios de Pagamento)
```
1. Login https://www.mercadopago.com.br
2. Sidebar → RECEBEDORES 
3. \"Meios de pagamento\" ou \"Formas de recebimento\"
4. Ativar:
   ✅ PIX
   ✅ Cartão Crédito/Débito
   ✅ Boleto
   ✅ Dinheiro em Conta MP
```

## 2. VERIFICAÇÃO DA CONTA (OBRIGATÓRIO)
```
Dashboard → CONFIGURAÇÕES → Verificação da conta
✅ CPF/CNPJ 
✅ Foto RG/IE
✅ Comprovante endereço
✅ Conta bancária (para receber)
✅ Aprovação 24-72h
```

## 3. Conta Bancária
```
CONFIGURAÇÕES → \"Contas bancárias\"
Adicionar banco PF/PJ para saques
```

## 4. Teste
```
Modo anônimo browser
Compre R$1 teste
PIX QR + lista cartões aparecem ✅
Saldo MP disponível imediatamente
```
